프로젝트 명 : chap13_javafx

 [프로젝트에 라이브러리 추가] 
 - 현재 프로젝트에서 javafx 클래스 사용을 위해서 라이브러리 추가  
 - project 오른쪽 메뉴 -> Properties
   -> Java Build Path -> Libraries 탭
   -> Add External JARs 버튼 클릭
   -> java\jdk1.8.0_91\jre\lib\ext 폴더에서 jfxrt.jar 선택
 
 - 이클립스 -> preference -> javafx -> SceneBuilder executable 지정(SceneBuilder가 깔린 곳에서 exe 연결)
